# 添加友链

```
{
  "title": "Akula<>Blog",
  "link": "https://lolicon.akulak.icu/",
  "img": "https://lolicon.akulak.icu/img/avatar.webp",
  "des": "苟利国家生死以 岂因祸福避趋之"
}
```
大佬想要添加友链的也可以按照上面这种方式留言 (主要是懒 方便我复制粘贴)
没有描述的话就默认是 名字 + "大佬"
熟人会自己根据印象添加描述()
